# Twitter2Gif

A minimalist, privacy-focused tool designed to preserve ephemeral digital media.
Archive videos, capture GIFs, and extract high-quality images from Tweets without watermarks or tracking.

## Features
- **Video to GIF**: Convert Twitter videos to optimized GIFs.
- **Video Download**: Download raw MP4 videos.
- **Image Support**: Download single or multiple images from tweets.
- **Privacy Focused**: No tracking, local proxy for privacy.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS (Ultra-Minimalist), JavaScript
- **Backend**: Node.js, Express, FFmpeg
